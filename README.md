# AI-AGENT: Professional Multilingual AI Chatbot

A powerful AI chatbot that converses in 40+ languages including regional variants. Professional distribution for end-users.

## Features

- **40+ Languages Supported** - English, Spanish, French, German, Hindi, Tamil, Telugu, Bengali, Arabic, Chinese, Japanese, and more
- **Automatic Language Detection** - Responds in your language automatically
- **Smart Conversations** - Maintains context and understands follow-up questions
- **Knowledge Base** - Answers questions on science, technology, history, arts, and more
- **Easy to Use** - Simple command-line interface, no coding required

## Quick Start

### Step 1: Download
```bash
git clone https://github.com/Joydip-hub/AI-AGENT.git
cd AI-AGENT
```

### Step 2: Setup
```bash
python setup.py
pip install -r requirements.txt
```

### Step 3: Run
```bash
python chat.py
```

That's it! Start chatting in any language.

## Available Commands

### Simple Version (chat.py):
- `help` - Show commands
- `stats` - View statistics
- `clear` - Clear history
- `quit` - Exit

### Advanced Version (main.py):
- `/help` - Show commands
- `/stats` - View statistics
- `/clear` - Clear history
- `/export` - Save conversation
- `/lang` - Show language
- `/quit` - Exit
